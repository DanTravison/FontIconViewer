# FontIconViewer

A simple WPF app for viewing and searching Segoe MDL2 Assets and Segoe Fluent Icons fonts.

Use search to match symbol names containing the search text.

# Building the project
Open the FontIconViewer.sln solution file in Visual Studio 2019 or Visual Studio 2022 and build the solution.

# Dependencies
> .Net 5 and uses 
> VirtualizingWrapPanel nuget package.
